# Cognitive Systems Glossary

## Architectures
- **CoALA**: Cognitive Architectures for Language Agents. Standardizes Memory, Action, Decision.
- **ACT-R**: Adaptive Control of Thought-Rational. Cognitive theory of learning.
- **OODA**: Observe, Orient, Decide, Act. Decision cycle for dynamic environments.

## Memory Terms
- **Working Memory**: Current context window (RAM).
- **Episodic Memory**: Past experiences (Hard Drive).
- **Semantic Memory**: Facts and world knowledge.
- **Procedural Memory**: Skills and tool usage rules.

## Concepts
- **Metacognition**: The agent's ability to think about its own thinking (e.g., "I don't have enough info").
- **Grounding**: Anchoring responses in retrieved facts (RAG) to prevent hallucination.
- **Alignment**: Ensuring agent goals match human intent.
