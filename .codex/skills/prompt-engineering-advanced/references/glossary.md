# Prompt Engineering Glossary

## Techniques
- **Zero-Shot**: Prompting without examples.
- **Few-Shot**: Prompting with 1-5 examples.
- **CoT**: Chain-of-Thought (reasoning steps).
- **ToT**: Tree-of-Thought (multiple paths).

## Concepts
- **Hallucination**: Model generating false information confidently.
- **Grounding**: Using external data (RAG) to verify facts.
- **Context Window**: The limit of text the model can see at once.
- **Token**: The basic unit of text for LLMs (~0.75 words).
- **Temperature**: Parameter controlling randomness (0.0 = deterministic, 1.0 = creative).

## Frameworks
- **RICE**: Role, Instructions, Context, Examples.
- **CRISPE**: Capacity, Role, Insight, Statement, Personality, Experiment.
