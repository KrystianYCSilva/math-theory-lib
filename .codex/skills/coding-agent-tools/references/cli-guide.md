# AI CLI Guide & Documentation

A detailed breakdown of the major AI coding assistants using verified documentation sources.

## 1. Claude Code (Anthropic)
-   **Capabilities**: Advanced reasoning, massive context window, native MCP support.
-   **Best For**: Complex architectural refactoring, reading massive documentation.
-   **Documentation**: [Claude Code Docs](https://code.claude.com/docs) | [Platform Docs](https://platform.claude.com/docs/pt-BR/home)

## 2. Gemini Code Assist (Google)
-   **Capabilities**: Fast inference, multimodal (text/image/video), deep integration with Google Cloud.
-   **Best For**: Rapid prototyping, multimodal tasks, large codebase retrieval.
-   **Documentation**: [Gemini API Docs](https://ai.google.dev/gemini-api/docs) | [Gemini CLI Skills](https://geminicli.com/docs)

## 3. GitHub Copilot
-   **Capabilities**: Workspace-aware completion, shell command explanation, CLI integration.
-   **Best For**: Daily coding, shell scripting, git operations.
-   **Documentation**: [GitHub Copilot Docs](https://docs.github.com/pt/copilot)

## 4. Cursor (AI Code Editor)
-   **Capabilities**: AI-native fork of VS Code. "Composer" mode for multi-file edits.
-   **Best For**: The primary IDE for AI-assisted development.
-   **Documentation**: [Cursor Docs](https://cursor.com/docs) | [Cursor Agents](https://cursor.com/agents)

## 5. Qwen (Alibaba)
-   **Capabilities**: Strong performance in coding benchmarks, often open weights.
-   **Best For**: Local inference, specialized coding tasks.
-   **Documentation**: [Qwen Code Docs](https://qwenlm.github.io/qwen-code-docs/)

## 6. Open Source Alternatives
-   **OpenDevin**: [GitHub](https://github.com/OpenDevin/OpenDevin)
-   **Aider**: [Website](https://aider.chat)
-   **Continue**: [Website](https://continue.dev)
