# Markdown Tools Ecosystem

## Editors & IDEs
- **VS Code**: Best all-rounder. Extensions: "Markdown All in One", "Markdown Preview Mermaid Support".
- **Obsidian**: Knowledge base on local Markdown files.
- **Typora**: Seamless live preview editor.

## Converters
- **Pandoc**: The "Swiss-army knife". Converts Markdown to PDF, Docx, HTML, LaTeX, Epub.
  `pandoc input.md -o output.pdf`
- **Marp**: Create slide decks from Markdown.

## Static Site Generators (SSG)
- **Jekyll**: GitHub Pages default.
- **Hugo**: Fastest SSG. Go-based.
- **Docusaurus**: React-based, great for documentation sites.
- **MkDocs**: Python-based, simple and elegant (Material theme).

## Linters
- **markdownlint**: Style checker. Enforces headers, spacing, and list rules.
