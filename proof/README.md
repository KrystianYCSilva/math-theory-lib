# proof

Proof objects and verification for Wave 4.

## Overview

This module provides:

- `Proof` — Sealed hierarchy of proof objects
- `ProofChecker` — Verifies proof validity
- `TheoremRegistry` — Stores proven theorems

## Notes

- Proofs are first-class objects that can be serialized
- Proof checker verifies each inference step
